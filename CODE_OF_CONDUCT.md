# Code of Conduct

Mojibake is committed to providing a welcoming, respectful, and harassment-free community for
everyone. Writing software is fun, and it should be for everybody.

1. Be respectful
2. Be collaborative
3. Respect differing viewpoints

Harassment, discrimination, threats, personal attacks, and other disruptive behavior are not
acceptable in any project space.

If you experience or witness unacceptable behavior, report it privately at `mojibake@zaerl.com`.
Reports will be handled as confidentially as reasonably possible.
